The following folder contains the code using for multidomain based on image recognition for MAPDH.
There are 2 latest versions of the code:
- Image_color_recognition_patterning_multi_domain_V12.py (used for multidomain patterning)
- Single_domain_grid_patterning (used for single domain patterning)
and a colorful digital mask using in the experiment.

Older version of the code are in the folder (Previous MAPDH code)

Other image masks are in the folder (Alternative digital masks)

Code was based and adapted for the MAPDH system (https://journals.plos.org/plosone/article?id=10.1371/journal.pone.0295923)

Code was done in Python 3.9.19. All library used were updated until their latest version in 2024.
